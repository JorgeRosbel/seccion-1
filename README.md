### Prueba Técnica | Sección 1 - React


### 1. Clone the repository
```bash
git clone https://github.com/JorgeRosbel/seccion-1.git
```

### 2. Change Directory
```bash
cd seccion-1
```

### 2. Install and Run
```bash
cd seccion-1

# npm
npm install
npm run dev

# Yarn
yarn install
yarn dev

# pnpm
pnpm install
pnpm dev
```
